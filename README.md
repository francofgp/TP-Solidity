# Ejecutar el proyecto


```bash
git clone https://github.com/francofgp/TP-Solidity.git

cd TP-Solidity

npm install

npm run build

npm test

npx hardhat run scripts/deploy.ts --network rinkeby

```

*Tuve que usar la version de solidity 0.8.1 y no la 0.7.3 que estaba en la guía*
